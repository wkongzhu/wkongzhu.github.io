算法描述尽可能用利于数字设计的方式，而不是程序设计的角度。

# 方法1

不妨设 $a > b$
初始化：
$$
r(-1) = a; r(0) = b; 
c(-1) = 1; c(0) = 0;
d(-1) = 0; d(0) = 1；
flag(0) = 1
$$
	
迭代过程：从 *n=1* 开始
$$
q = r(n-2)/r(n-1)
r(n) = r(n-2) - q * r(n-1)
c(n) = c(n-2) + q * c(n-1)
d(n) = d(n-2) + q * d(n-1)
flag(n) = -1 * flag(n-1)
x(n) = flag(n-1) * c(n)
y(n) = flag(n) * d(n)
$$

结束条件： r(n) == 0
输出： 
$$
r(n-1), x(n-1), y(n-1)
gcd(a, b) = r(n-1)
x(n-1) * a + y(n-1) * b = r(n-1)
$$
